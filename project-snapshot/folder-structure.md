- `apps/`: npm workspace applications. Currently contains `backend` and an empty `dashboard` directory.
- `apps/backend/`: the implemented Node/TypeScript Express app and static dashboard assets.
- `apps/backend/src/`: backend source code.
- `apps/backend/src/airtable/`: low-level Airtable client, schema map, and internal-field adapter.
- `apps/backend/src/config/`: environment loading and runtime configuration.
- `apps/backend/src/connectors/`: connector framework, mock connectors, registry, and Spotify CSV connector.
- `apps/backend/src/ingestion/`: reusable CSV parsing and Spotify CSV transformation.
- `apps/backend/src/logging/`: structured console logger.
- `apps/backend/src/services/`: business logic, dashboard aggregation, KPI scoring, Mailchimp sync, and sync orchestration.
- `apps/backend/src/services/platformAnalytics/`: channel-level metric normalizers for non-Spotify channels.
- `apps/backend/src/types/`: shared TypeScript domain and Airtable table types.
- `apps/backend/public/`: current dashboard UI. This is static browser code, not React.
- `docs/`: earlier architecture and migration documents. Some are now stale relative to recent Spotify changes.

- `scripts/`: documentation tooling, including this generator.
