## Important Query Parameters

- `/api/overview`: `startDate`, `endDate`, `dateMode`
- `/api/channel-breakdown`: `startDate`, `endDate`, `dateMode`
- `/api/top-content`: `timeframe`, `platform`, `groupBy`
- `/api/comparative`: `period`
- `/api/integrations/mailchimp/sync`: `periodType`; requires sync auth when `SYNC_API_KEY` is configured
- `/api/ingestion/sync`: multipart/raw upload with `connectorId`; Spotify requires CSV file payload

## Response Shapes

Dashboard endpoints return plain JSON objects. The frontend expects:

- `/api/overview`: score definitions, monthly activity, growth indicators, monthly snapshot, charts, source status.
- `/api/top-content`: sections, topOverall, platforms, groups, scoreFormula.
- `/api/channel-breakdown`: `dateRange`, `previousDateRange`, `summary`, `channels[]`, `trends.channels[]`.
- `/api/status`: `sources[]`.
