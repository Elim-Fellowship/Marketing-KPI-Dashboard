## Service Notes

- `CommunicationsAnalyticsService`: top-level dashboard query service. Spotify Channel Breakdown is now built directly from `spotifyEpisodeMetrics`.
- `DashboardAggregationService`: aggregates non-Spotify channel metrics from email, social, website, and future placeholder channels.
- `MailchimpService`: live Mailchimp API integration and `KPI_History` writer.
- `SyncManager`: generic connector orchestrator with in-memory status/history.
- `AirtableService`: cached read facade over `AirtableClient`.
- `AirtableAdapter`: write-time schema translator between internal names and Airtable field names.
- `kpiCalculationEngine`: scoring formulas and percent-change utilities.
- `communicationsIntelligenceModel`: dashboard model builder for KPI overview, top content, comparative analysis, and content scores.
